# VMM

The `vmm` is a Virtual Machine Manager designed to manage the lifecycle of micro-VMs. It provides low-level abstractions over KVM (Kernel-based Virtual Machine) and handles the creation, configuration, and management of virtual machines.

## How to Start

1. Ensure you have Rust installed on your system.
2. Navigate to the `vmm` directory:
   ```bash
   cd vmm
   ```
3. Build the VMM:
   ```bash
   cargo build
   ```

For detailed documentation, refer to [docs/vmm.md](../docs/vmm.md).
